# XML 파서
## 소개
XML 파서는 자바 언어를 기반으로 XML 파일을 읽고 파싱하여 DomTree 객체로 변환하는 애플리케이션입니다.

## 빌드 및 실행
```shell
~ $ gradlew build
~ $ java -jar build/libs/xml-parser-1.0-SNAPSHOT.jar index.xml 
```
- index.xml는 빌드 및 실행의 예시 파일입니다.
- 실행시 원하는 xml 파일 제목 및 확장자를 index.xml 대신 넣으시면 됩니다.
